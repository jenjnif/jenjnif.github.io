My desire to become a programmer developed whilst volunteering as a molecular biologist and working for a specialist Global Health web development company based in London. I started self-teaching myself various aspects of programming and enjoyed it so much that in June 2017 I left my job to seriously pursue this path.

I have been working as a Python programmer since April 2018 alongside which I immersed myself in many books, tech meetups and (mainly free) courses. For those interested, below is a list of the main courses and resources I have used.

Useful resources

HackerRank, a great site to test your skills

I heard of HackerRank through a friend and am really enjoying working through the challenges. This is a good site to work on your programming skills and to practice the kind of tests you might get from a potential employer. I will be completing more challenges in the near future and perhaps my first contest. Watch this blog space!

Completed courses

Try SQL, SQL, Codeschool (online), completed 27 April 2018

Try Docker, Docker introduction, Codeschool (online), completed 5 March 2018

Gives a good introduction to the basics of Docker. How to use images to build Docker containers and configure the images with Dockerfiles in the command line. The next step is to build a Node App with Postgres and Docker.

Flying Through Python, Python, Codeschool (online), completed 3 April 2018

One of the things I find really useful about Codeschool is the screencasts that you can follow and work through once you have completed the courses. Each one is roughly an hour long and works through a project so you can consolidate all the knowledge that you just learnt in a useful way. The Python one was no exception and within a couple of hours I had built a Weather Forecast Mailer, which can be viewed on my GitHub account.

Front-End Foundations, HTML and CSS, Codeschool (online), completed 3 March 2018

Programming for Everybody (Getting Started with Python), Python, University of Michigan, coursera (online), completed 30 December 2017

This course was a great introduction to python with the view to working with data. It was based on the book "Python for Everybody" by Charles Severance, which I am reading to continue my learning. Afterwards I intend to read the book on which it is based, "Think Python" by Allen Downey.
